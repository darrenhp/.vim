vundle
======

my-vim-config based on vundle

- 下载配置    : `git clone https://github.com/darrenhp/vundle.git`
- 链接配置文件: `ln ~/.vim/vimrc ~/.vimrc`

- 安装Vundle：`git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle`
- 安装插件： `:BundleInstall`
- 更新插件： `:BundleInstall!`
- 卸载不在列表中的插件： `:BundleClean"`
